#Dog Api Scraper App

##Description

An app to scrape data from the https://dog.ceo/dog-api/ Dog Pictures API and add it to a database.

Designed to be used with the NMR Top-dog App.

##Installation

* Clone git repo on local machine. 
* Setup database to receive info:
